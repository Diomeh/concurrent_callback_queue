Example
const queue = new ConcurrentCallbackQueue({
    autoStart: false,
    maxConcurrent: 2,
    onCallbackError: (error) => console.error(error),
});

for (let i = 0; i < 10; i++) {
    queue.enqueue(() => {
        const uri = 'https://httpstat.us/200,400?sleep=2000';
        return $.get(uri).done(() => {
            console.log(`Request ${i+1} completed`);
        });
    }, 3);
}

queue.start();

start()

Inicia la ejecución de la cola.

Inicia la ejecución de la cola.

stop()

Detiene la ejecución de la cola, pero no elimina los callbacks pendientes de ejecución.

Detiene la ejecución de la cola, pero no elimina los callbacks pendientes de ejecución.

Llamar a este método no detendrá la ejecución de los callbacks que ya se están procesando, tampoco se eliminan los callbacks pendientes de ejecución, por lo que si se vuelve a iniciar la cola, se reanudará desde el último callback que estaba pendiente.
